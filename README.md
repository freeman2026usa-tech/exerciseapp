## 身体注意事项（重要，已编入计划）

- **左膝骨性突起**：疑似奥斯古德病（胫骨结节骨骺炎）残留骨突，通常无痛不影响。跪姿动作垫软垫别硬顶，避跳跃/深屈膝到疼；若痛/肿就医。
- **肋骨外翻 + 探头（圆肩头前引）**：靠上背/肩后束强化 + 核心抗伸展 + 呼吸控制改善。
- **双侧肘超伸（左更明显）+ 深位左肘弹响**：推类动作用**中立握**、肘收约 45°、**软肘不锁死**、控制深度。无痛可练，痛则就医。
- **左后腰紧绷 + 弹响髋（内侧/髂腰肌）**：同一根源 = 髋屈肌紧 + 臀弱 + 骨盆前倾（和肋骨外翻是一套）。放松髂腰肌、强化臀、稳核心。无痛良性，痛/卡/影响走路就找康复科。
- **热身扩胸时肩弹响**：热开后减少 = 良性。渐进热身、先激活肩胛、不弹震。
- **俯卧撑底部憋气致头胀**：不憋气，呼气发力，用 360°呼吸稳核心代替憋气。

> 以上为一般健身指导，非医疗建议。明显疼痛请停止并咨询专业人士。

## 目标拆解（倒三角 = 上宽下窄）

1. **肩中束**（侧平举）→ 宽度关键
2. **背阔 + 上背**（引体、划船）→ V 型轮廓
3. **胸**（俯卧撑进阶、哑铃卧推）→ 厚度
4. **核心 + 体态**（改善肋骨外翻/探头/骨盆前倾）→ 让胸廓更挺、比例更好

## 器械评估

- **现在买**：力量带（辅助引体）、臀圈、泡沫轴、软尺、加厚跪垫、体重秤 —— 规格见 [Stage0_准备.md](Stage0_准备.md)

## 训练思路（已确认骨架）

短课高频：每次 20–40 分钟、可天天练、一周约 5–6 天。**四张菜单滚动循环**：

- **A 推**（胸 + 三头）
- **B 拉**（背宽 + 二头）— 频率高
- **C 肩 + 体态**（肩中束/后束 + 核心矫正）— 频率高
- **D 腿 + 核心**（哑铃护膝，维持）— 频率低

顺着练、循环走，休息随时插入不占名额。每张菜单的详细动作卡见下方各菜单文件。
