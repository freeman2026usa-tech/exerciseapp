# 健身伴侣工具（训练中引导播放器）

把 `exerciseapp/` 里的私教手册变成**训练时能用**的伴侣：语音优先、大字卡、组间自动计时、一键记录、自动算下次目标。**离线、无需安装、无需联网。**

## 怎么打开

直接双击 `app/index.html`，用 **Chrome 或 Edge** 打开即可（电脑大屏最佳，手机也能用）。
> 第一次点「开始」或勾热身时会触发语音——浏览器要求用户先有一次点击才允许发声，属正常。

## 听不到语音？（重要）

语音靠**系统里的中文语音引擎**发声。主页有一条状态：
- **✅ 中文语音就绪** → 直接能用，点「测试语音」验证。
- **⚠ 未检测到中文语音** → 你的 Windows 没装中文语音（默认常只有英文），所以静音。点主页「如何安装」按步骤装一个中文语音，**装完重启浏览器**即可。也可以用 **Edge**（自带更好听的在线中文语音）。
- 没语音也不影响练：**大字 + 组间提示音**（提示音不需要语音包）照常工作。

## 怎么用（一次训练的流程）

1. **主页**：顶部「今日建议」已按 A→B→C→B→C→D 循环自动接上一次。点大卡 `▶ 开始今日训练`，或手动选 A/B/C/D。
2. **热身**：7 步清单，逐条点一下打勾（会语音读出），点 `进入正式训练`。
3. **逐组引导**：大字显示动作名 / 第几组 / 目标组次·节奏·重量 / 一条护伤关键提示，**下面直接是「怎么做」编号步骤 + 该有/不该有的感觉**（不用点开就能照做），并语音播报。
   - `✔ 完成本组` → 弹出记录（次数/重量步进器，已预填上次或目标值）→ 保存。
   - `📖 详情/编辑`：展开完整动作卡（起始姿势、常见错误、退/进阶），并可**编辑**这个动作（见下）。
   - `太难→退阶` / `太易→进阶`：语音+文字给你替代方案。
4. **组间休息**：大号倒计时 + 进度环，「还有十秒」语音 + 提示音，结束自动提示下一组。可 `+30s` 或 `跳过`。
5. **总结**：本次动作/组数一览 → 填短板打卡（俯卧撑/引体最多连续次数）、体重、状态 → `保存并结束`。循环自动前进，下次进来接上下一张。

## 数据存哪 / 怎么备份

- 全部数据存在**浏览器本机**（localStorage，键 `exerciseCompanion.v1`）：训练日志、每个动作上次的次数重量（用于进阶预填）、循环位置、设置。
- **换电脑/清缓存前先备份**：主页 `导出备份` 下载一个 JSON；到新环境用 `导入备份` 恢复。
- 只清这个网站的站点数据会清空记录 → 记得先导出。

## 进阶怎么算（自动）

按手册「双重渐进」：先加次数，各组都到区间上限后再加难度。
- 你现在只有一对 **10kg 固定哑铃**，所以哑铃动作「加重量」这条暂走**放慢离心 / 顶端多停**替代（等俯卧撑 15–20、引体 5 后再买可调哑铃/更重的）。
- 徒手动作（俯卧撑/引体/派克）各组都到上限 → 提示升「阶梯」下一档。

## 改内容（两种方式）

- **App 内直接改（推荐，不碰代码）**：任一动作 → `📖 详情/编辑` → `✏ 编辑`，可改**组数 / 次数 / 节奏 / 组间歇 / 默认重量 / 关键提示**，点保存**即改即生效**，改动存在你本机（随 JSON 备份一起走），随时可`恢复默认`。
- **改默认值 / 长文说明** → 改 `app/program.js`（数字与出厂默认）或对应 `菜单X.md`（长篇说明）。`program.js` 是 App 的出厂数据，你在 App 里的编辑是叠加在它之上的个人覆盖。

## 训练记录导出（回流到别处管理）

- **练完自动弹出**一段 Markdown 记录（与《训练日志.md》同格式），可`复制`到 Notion/Obsidian，或`下载 .md`。
- 事后也能补：主页`训练日志` → 每条右下角 `复制 Markdown` / `下载 .md`。
- 想并进项目里的 `训练日志.md`：把这段贴到「记录区」最上面即可。（以后 P4 可做成 App 自动写回文件。）
