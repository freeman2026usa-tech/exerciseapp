# 健身伴侣工具（训练中引导播放器）

把私教手册变成**训练时能用**的伴侣：语音优先、大字卡、自动计时、一键记录。**离线、无需安装、无需联网。**

分两个板块，各自独立开始与记录：
- **💪 力量训练**：四菜单 A→B→C→B→C→D 循环，逐组引导 + 自动算下次目标。
- **🌙 夜间放松**：睡前 10 分钟 E1–E4（腹式呼吸 / 反向凯格尔 / 快乐婴儿式 / 深蹲停留），逐步计时 + 呼吸动画 + 轻量打卡。

## 怎么打开

直接双击 `app/index.html`，用 **Chrome 或 Edge** 打开即可（电脑大屏最佳，手机也能用）。
> 第一次点「开始」或勾热身时会触发语音——浏览器要求用户先有一次点击才允许发声，属正常。

## 听不到语音？（重要）

**分层音频引擎**（内容 / 渲染 / 编排解耦）：
- **呼吸用提示音（闭眼也懂）**：夜间放松用**吸气上行长音、呼气下行长音**打节拍——升调=吸、降调=呼，一听就分得清；进第一步先语音引导一句「跟着提示音走，升调吸气，降调呼气」。**语音只在呼气间隙随机念一句要点**，不聒噪、和提示音不打架。
- **语音音质**：`声音`下拉选中文语音、可`试听`，选定即记住。**想好听 → 用 Edge 打开**，会出现"晓晓"等自然语音（联网、接近真人、免费）。没中文语音也能练（提示音+大字照常）；点`如何安装`可装。
- **背景音乐**：`音乐`下拉选——**默认关闭**；内置「轻音·合成钟琴」（现场合成的柔和氛围，非白噪音）；或**放你自己的 mp3**（进 `app/audio/music/`、按 `tracks.js` 里说明加一行即可）。说话时自动压低。
- **音量分开调**：语音 / 音乐 / 提示音 三个滑块各自独立。
- **锻炼提示词**（主页 `🗣 锻炼提示词`）：看得见、能改所有提示——每个动作的**语音提醒**和**屏幕关键提示**、夜间每步的**一句提示**和**要点池**；改完即用、无需重生成，可一键恢复默认。
- 力量训练时语音只报一句**行动提醒**（如"呼气推起，软肘别锁死"），完整要点看屏幕。

> **声音怎么渲染 / 以后换大模型**：出声走唯一入口 `say()`——先查预渲染音频 clip（`app/audio/manifest.js`，现为空）→ 可插拔运行时 `window.AI_TTS`（默认无）→ 浏览器实时 TTS 兜底。以后想换"大模型渲染的好声音"，**只跑个生成脚本产出 clip、App 代码不改**：推荐**预渲染**（edge-tts 免费 / OpenAI / ElevenLabs 任选，密钥只在你机器生成时用、不进 App，离线、跨浏览器一致）。

## 怎么用 · 力量训练

1. 主页点 **💪 力量训练** → 顶部「今日建议」已按 A→B→C→B→C→D 循环接上一次，点 `▶ 开始今日训练`，或手动选 A/B/C/D。
2. **热身**：7 步清单（可逐条点，也可直接 `进入正式训练`）。
3. **逐组引导**：大字显示动作名 / 第几组 / 目标 / 一条护伤提示，下面直接是「怎么做」步骤 + 该有/不该有的感觉。
   - **默认全自动（少点点）**：`✔ 完成（按计划）` 一键按目标记录并自动进休息 → 休息结束自动进下一组 → 一个动作做完自动进下一个动作。想改数字点 `改一下`。
   - 想每步自己掌节奏：主页设置里关掉 `⚡ 自动进行`，换动作时会停下等你确认。
   - `📖 详情/编辑` 看完整卡并可编辑；`太难/太易` 给替代方案。
4. **总结**：动作/组数一览 + 短板打卡 + 体重/状态 → `保存并结束`，循环自动前进。

## 怎么用 · 夜间放松

1. 主页点 **🌙 夜间放松** → 直接开始 E1。
2. **闭眼纯听觉**：屏幕极简（只步骤名 + 倒计时 + 按钮 + `做法（睁眼看）`）。引导全靠声音——**升调=吸气、降调=呼气**的提示音打拍子（E1 吸 4 秒/呼 6 秒），呼气间隙随机念一句要点；到点**自动进下一步**。可 `⏸ 暂停`、`+30s` 或 `跳过这步`（E4 深蹲膝不适可直接跳）。
3. 做完 → 轻量打卡：`做了吗` / `反向凯格尔找到「松」的感觉` / 备注 → 保存。

> 训练/放松时**屏幕自动常亮**（不熄灭），组间休息和夜间每步都能`⏸ 暂停`。
> 前 4 周只放松、不练收紧（盆底偏紧，收紧会更糟）。做完直接睡、别碰手机。

## 数据存哪 / 怎么备份

- 全部数据存在**浏览器本机**（localStorage，键 `exerciseCompanion.v1`）：训练日志、每个动作上次的次数重量（用于进阶预填）、循环位置、设置。
- **换电脑/清缓存前先备份**：主页 `导出备份` 下载一个 JSON；到新环境用 `导入备份` 恢复。
- 只清这个网站的站点数据会清空记录 → 记得先导出。

## 进阶怎么算（自动）

按手册「双重渐进」：先加次数，各组都到区间上限后再加难度。
- 你现在只有一对 **10kg 固定哑铃**，所以哑铃动作「加重量」这条暂走**放慢离心 / 顶端多停**替代（等俯卧撑 15–20、引体 5 后再买可调哑铃/更重的）。
- 徒手动作（俯卧撑/引体/派克）各组都到上限 → 提示升「阶梯」下一档。

## 改内容（两种方式）

- **App 内直接改（推荐，不碰代码）**：任一动作 → `📖 详情/编辑` → `✏ 编辑`，可改**组数 / 次数 / 节奏 / 组间歇 / 默认重量 / 关键提示**，点保存**即改即生效**，改动存在你本机（随 JSON 备份一起走），随时可`恢复默认`。
- **改默认值 / 长文说明** → 改 `app/program.js`（数字与出厂默认）或对应 `菜单X.md`（长篇说明）。`program.js` 是 App 的出厂数据，你在 App 里的编辑是叠加在它之上的个人覆盖。

## 训练记录导出（回流到别处管理）

- **力量**：练完自动弹出一段 Markdown（与《训练日志.md》同格式），可`复制`/`下载 .md`；事后在主页`训练日志`里每条也能再导出。
- **夜间**：打卡后弹出一行（与《康复日志.md》放松打卡同格式：`日期：晚间放松 做/没做 · 反向凯格尔找到感觉? 是/否/部分 · 备注(…)`）；主页`夜间记录`里可再导出。
- 想并进项目里的 `训练日志.md` / `康复日志.md`：把对应文本贴到「记录区」最上面即可。（P4 可做成自动写回文件。）
