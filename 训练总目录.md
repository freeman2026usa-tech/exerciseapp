## 一、怎么读动作卡

每个动作卡都有固定几栏，照着看：

- **组×次**：如「4 × 8–12」= 4 组、每组 8–12 次。
- **RIR**：留几次余量。「RIR2」= 做到"还能再做 2 个就停"。**新手别练到力竭。**
- **节奏 a-b-c**：下放 a 秒 - 底部停 b 秒 - 发力 c 秒。「3-1-1」= 慢下 3 秒、底停 1 秒、1 秒发力。
- **组间歇**：组与组之间休息秒数。
- **该有/不该有的感觉**：练对了肌肉在哪、哪种感觉是危险信号。
- **常见错误 / 易踩的坑 / 自查**：练之前先看，练错了能自己发现。
- **太难→退阶 / 太易→进阶**：动作太难或太简单时怎么换。

## 二、四菜单滚动循环

把训练当四张菜单，顺着 A→B→C→D 练，练完从头再来。**想休息哪天就休息，不占名额，下次接着没练的那张继续。**

| 菜单 | 主题 | 频率倾向 |
|---|---|---|
| **A** | 推（胸 + 三头） | 中 |
| **B** | 拉（背宽 + 二头） | 高 |
| **C** | 肩 + 体态 | 高 |
| **D** | 腿 + 核心（护膝） | 低（维持） |

**建议顺序**（偏重背肩，倒三角的关键）：A → B → C → B → C → D → 回 A……

**示例一周**：
> 周一 A推 · 周二 B拉 · 周三 C肩 · 周四 休息 · 周五 B拉 · 周六 C肩 · 周日 D腿 → 下周一 A推……

这样一周背、肩各摸两次，胸一次多一点，腿一次维持。

**每次结构**：热身 5–8 分钟（见 [热身与地基.md](热身与地基.md)）→ 当天菜单 3–4 个动作。总时长 20–40 分钟。**时间紧就砍每张菜单最后一个辅助动作。**

## 三、护伤总则（每次都记着）

这几条是根据你的身体情况定的，贯穿所有动作：

1. **推类中立握 + 软肘**：俯卧撑/卧推用掌心相对的握法，肘收到离身体约 45°，推到接近伸直就停、**不锁死**。护你的双肘超伸和左肘弹响。
2. **全程不憋气**：呼气发力、底部不停顿闭气。防你之前俯卧撑做完的头胀。
3. **大幅度扩胸前先激活肩胛**（弹力带撕裂），不做弹震式甩动。减肩弹响。
4. **下肢不跳跃、不深屈膝到疼**；左膝骨突处跪姿垫软垫。
5. **分清"酸胀"和"疼痛"**：肌肉酸胀正常；关节刺痛、异响伴痛、麻 → 停，告诉我换动作。
6. 每次开练前的**臀圈激活 + 泡沫轴放松髂腰肌**别省，这是治你弹响髋、腰紧、骨盆前倾的日常剂量。（泡沫轴未购前用弓步髋屈肌拉伸替代，见 [热身与地基.md](热身与地基.md)）
7. **收核心 ≠ 夹会阴**：发力时收肋骨、收核心，但**别顺带夹紧会阴 / 盆底**，每组之间主动松一下。你盆底偏紧、又在做早泄康复，硬夹着等于把它练得更紧、抵消晚上放松的功。

## 四、两个短板的进阶阶梯（重点盯）

**俯卧撑**（你现在标准约 12 个）：每档做到 **3×12 干净次数**就升下一档
1. 中立握标准俯卧撑 ← *你在这*
2. 慢离心（下放 4 秒）
3. 脚垫高俯卧撑（decline，练上胸）
4. 更难变式 / 单侧加负
5. 负重（之后配训练凳/背包）

**引体**（现在 2 个，先冲连续 5 个）：
1. 悬垂静力：死握 20–30 秒 ×几组
2. **辅助引体**（力量带）/ **离心引体**（4–5 秒慢放）← *当前主练*
3. 半程引体
4. 全程引体逐步加次数
5. 到 8+ 后再加负重

> 每次把这两项实际次数发我，我决定何时升档。

## 五、渐进超负荷（怎么变强）

**双重渐进**：先加次数，到区间上限再加难度/重量。
- 哑铃动作：区间内每次尽量多做 1 次；所有组到上限（如 3×12）→ 下次加重量或放慢离心。
- 徒手动作：达标 → 按上面阶梯升变式。
- 每周同动作比上周多 1–2 次总次数，就是进步。

**调整节奏**：每 3–4 周小复盘（看日志/围度/照片）；每 8–12 周阶段升级（短板过关后换新计划、考虑添训练凳）。

## 六、怎么记录

每次练完，把每个动作的**实际组数×次数 @重量**和感受发我，例：

> 「A1 中立握俯卧撑 4 组 10/9/8/8；A2 地板卧推 3×10@10kg；A3 飞鸟 3×12@5kg。左肘不响了，膝无痛，最后有点累。」
